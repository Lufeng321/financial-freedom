## 1. 今日视图 (`index.html`)
定位：整合日常核心关注（任务/习惯/财务/运动）。

模块：
- 今日焦点：静态示例列表；勾选后样式变灰 (class `done`)。
- 习惯打卡快捷：按钮一次性打卡并禁用。
- 自由指数迷你条：横向进度条（`data-shape=bar`），显示日/年被动与支出指标。
- 运动概览：3 个指标 + SVG 进度圆（静态示例）。
- 悬浮操作 FAB：展开后可跳转或未来挂接动作（当前仅样式）。

数据来源：自由指数与财务指标来自 `prototype.js` 中 finance 模块运行时计算（依赖账单与资产数据）。

## 2. 记账 (`bill.html`)
定位：所有财务指标的基础输入入口。

表单字段：
- 收支类型 `kind`: in/out
- 收入性质 `incomeType`: active/passive（仅当收入时显示）
- 支出类型 `expenseType`: necessary/optional（仅当支出时显示）
- 分类 `category`: 根据 in/out 动态填充（两组数组）
- 金额 `amount`
- 日期时间 `datetime`
- 备注 `remark`
- 设为模板 `asTemplate` (尚未实现逻辑)

交互逻辑：
- 切换收支类型时：隐藏/显示对应性质/类型字段 + 重建分类下拉。
- 提交后：记录压入内存数组 `bills` -> 重绘最近记录（倒序取 20）。
- 触发财务指标（自由指数等）刷新。

## 3. 财务自由仪表盘 (`finance.html`)
定位：目标导向的核心可视化页面。

核心指标：
- 自由指数：日被动收入 / 日支出（顶 100%）。
- 年/日被动收入：被动账单 + 资产年化收益 / 365。
- 年/日支出：支出账单累计 / 365。
- 距离财务自由天数：基于差额与粗略增长模型估算（示意）。

资产与负债：
- 资产字段：name, amount, roi。
- 负债字段：name, amount, rate。
- 汇总：总资产、总负债、净资产、预计年被动收入(∑ amount * roi)。
- 添加：使用 `prompt` 输入（原型简化）。
- 编辑：点击列表项弹出多轮 `prompt` 直接修改（新增）。

计算器：
- 输入：年度支出 / 当前资产 / 预期回报率。
- 输出：目标核心资产 = 年支出 / 回报率；缺口；五年滚动计划（均匀投入 + 再投资）。

趋势区：三个图表占位（后续引入图表库）。

## 7. 反思日记 (`diary.html`)
定位：情绪追踪 + 反思文本 + 媒体补充。

功能：
- 日期输入。
- 情感评分：5 个表情点击激活（高亮）。
- 文本域：输入 500ms 防抖模拟保存状态（“保存中…”→“已保存”）。
- 图片上传：多选 `image/*` -> 生成本地 URL 预览 + 删除按钮，内存数组管理。
- 历史日记示例列表（静态）。

限制：无本地存储；刷新丢失；无图片压缩。

## 8. 运动记录 (`sport.html`)
定位：基于标准化 typeId 的运动数据采集，支持多种指标组合与后续同步拓展。

分类与指标映射：
- 100x / 200x（基础训练/休闲与球类）→ 指标: time + calorie
- 300x（耐力有氧）→ 指标: time + distance + calorie
- 400x（计次数）→ 指标: number + calorie

核心功能：
- 选择运动类型 (typeId) 后动态生成所需指标输入框 (范围校验: time 1-1440 分; distance 1-100000 米; number 1-10000; calorie ≥1)。
- 表单提交：创建记录 {typeId, 指标集合, datetime, syncWanted, synced:false} 并加入历史列表。
- 历史记录：最新优先倒序，显示阶段时间戳、指标 Pills、同步状态（待同步 / 已同步）。
- 同步模拟：聚合所有待同步记录构造 mock payload (recordList)，延迟 500ms 标记为已同步 + 推送消息中心。
- XP：每条运动记录提交奖励 XP（在全局经验系统中实现）。

待办增强方向：实际对接微信运动 / 健康平台 API、图表化展示各指标趋势、日/周负荷评估、恢复建议。

## 9. 技能管理 (`skill.html`)
定位：技能进度 + AI 辅助学习路径原型。

新增：AI 学习计划生成模块（前端模拟，可后续接入真实大模型接口）。

AI 计划输入：
- 想学习的技能 targetSkill
- 每日可投入时间 dailyMinutes
- 当前掌握情况 currentLevel (zero/basic/intermediate/advanced)
- 学习目标 goal (可选)

生成逻辑（simulateAIPlan）：
- 根据 currentLevel 设定基础周数 (12/10/8/6)，按每日时长系数 (≥120min:0.8, ≥60:1, 否则1.2) 得出估算学习周期 weeks。
- 阶段划分：基础认知25% / 结构深化30% / 实战项目30% / 巩固提升15%，各阶段生成 2-3 个任务 (title=阶段:焦点点)。
- 为“实战项目”首任务标记焦点(focus=true)。
- 每任务预估学习天数 est = weeks * 阶段百分比 * (dailyMinutes/60) *（基础首任务减半示例系数）。

结果展示：
- 摘要：技能名称 + 周期估算 + 强度标签 + 目标。
- 任务列表：复选框 + 优先级（实战>结构>基础/巩固），焦点任务标记。
- 全选/取消按钮；导入选中的任务到待办页 (`todoList`) 或首页焦点列表 (`focusList`)。
- 导入后推送消息中心反馈。

原有功能：技能进度条、+经验按钮（仍为示例）；成就徽章示例保留。

后续拓展：用真实模型产出里程碑/资源/评估题；任务生成可拆分为按周排程；与 XP / 成就联动（阶段完成奖励）。

## 10. 全局脚本 (`prototype.js`)
包含：
- FAB 展开逻辑。
- 番茄钟计时与 UI。
- 待办交互 (checkbox 完成 / 删除 / 焦点 / 编辑弹窗)。
- 习惯管理与日历、打卡 streak 计算。
- 日记自动保存、防抖、情感评分激活；图片上传预览与移除。
- 运动记录：typeId→指标映射 & 动态字段生成、范围校验、历史渲染、同步模拟、XP 奖励。
- 财务模块：
  - 资产/负债添加 + 点击编辑 (prompt 多步骤)。
- AI 学习计划：生成阶段任务、计划摘要、批量导入待办、全选/取消。
- 财务模块：
  - 记账表单动态字段显示与分类切换。
  - 账单列表渲染。
  - 资产/负债数据结构与添加、净资产与年化被动收益计算。
  - 自由指数与估算剩余天数、环形或条形两种形态渲染。
  - 财务自由计算器（目标资产 / 缺口 / 五年计划）。
  - 经验系统：番茄完成 / 习惯打卡 / 成就解锁 → 增加 XP，平方级经验曲线 (Lv^2 * 100)。
    - XP 来源扩展：记账（被动收入>普通收入>支出>非必要）、运动记录、技能增加经验按钮。
  - 成就判定：第一条习惯 / 连续7天 / 自由指数50% / 财务自由 / 5番茄。
  - 消息中心：XP、成就等事件推送到“我的”页面列表。

## 11. 当前已知缺口 / 待办建议（更新）
1. 数据持久化：localStorage/IndexedDB 或后端接口（所有核心数据结构）。
2. 模板记账：`asTemplate` 定义周期 / 金额规则 / 自动生成定时器。
3. 图表可视化：引入 ECharts / Chart.js 完成趋势、对比、分类占比；运动指标趋势与分布。
4. 待办：过滤/排序逻辑、任务持久化、导入去重、任务编辑统一组件化。
5. 运动：真实同步（WeRun / HealthKit）接口、周负荷与恢复指数、热力图。
6. 自由指数：更精细预测（月度现金流、增长率、通胀、安全边际>100% 区间着色）。
7. 日记：图片压缩、限额、情绪/关键词分析、搜索与情绪趋势图。
8. 导航：统一导航栏或侧边抽屉 + 路由化（SPA）。
9. 资产管理：资产类型枚举 + 默认年化、资产再平衡提醒、收益曲线。
10. AI 学习计划：接入真实 LLM，生成周排程/资源链接/评估节点；任务阶段完成自动加 XP。
11. 经验体系：不同来源加权、等级奖励、每日上限与防刷机制。
12. 无障碍 & 国际化：多语言、ARIA、日期本地化。

## 16. AI 学习计划逻辑补充
阶段划分与任务生成仅为启发式：
1. 依据当前水平设定基础周数；按每日投入时间修正周期。
2. 拆分 4 阶段（基础→结构→实战→巩固），按百分比分配学习“量”。
3. 每阶段焦点点列表映射为任务；设置优先级（实战=高）。
4. 允许用户勾选后导入待办（直接 DOM 注入——原型，不含去重与持久化）。
5. 后续可由 LLM 输出：
  - 更细时间粒度（每日/每周）
  - 资源链接（官方文档 / 教程 / 项目示例）
  - 阶段评估题或测验点
  - 自适应迭代（完成反馈后重排剩余计划）
